# AutoTest

## Envirioment:
  * Maven
  * JDK1.8
## Configuration:
  * src/main/config/application_dev.properties
  * src/main/config/application_test.properties
## How to run:
  1. Clone project 
  2. cd AutoTest
  3. mvn test -Pdev or mvn test -Ptest
